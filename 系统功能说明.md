# 翻译工具 — 系统功能说明

## 概述

Windows 桌面翻译工具，基于 PySide6 构建。支持迷你悬浮窗和完整窗口两种模式，集成 Google / 百度翻译引擎，Windows 系统 OCR 截图翻译、剪贴板监听和历史记录管理。

## 核心功能

### 1. 迷你翻译窗口（Ctrl+Shift+Q）

- 无边框悬浮窗，默认置于屏幕右下角
- 输入文字后 500ms 防抖自动翻译
- 支持拖拽移动、置顶切换（📌）
- 标题栏按钮：⚙ 设置、📌 置顶、✕ 关闭
- 底部按钮：历史、切换到大窗口、复制结果、清空
- 点击「切换到大窗口」同步当前文字到完整窗口

### 2. 完整翻译窗口（Ctrl+Shift+W）

- 标准窗口，支持最大/最小化
- 语言交换按钮（⇄）：快速交换原文和译文语言
- 再次按快捷键可隐藏窗口（切换模式）
- 切换回小窗口时保留当前文字

### 3. 截图翻译（Ctrl+Shift+E）

- 全屏选区模式，拖拽框选需要翻译的区域
- ESC 或右键取消选区，自动恢复之前的窗口状态
- 通过 Windows 系统 OCR 识别图片文字后自动填入迷你窗口并翻译
- **免费使用，无需配置**（需系统已安装中文语言包）

### 4. 翻译引擎

| 引擎 | 配置要求 | 说明 |
|---|---|---|
| Google 翻译 | 无需配置 | 通过 deep_translator 调用，始终可用 |
| 百度翻译 | 需 APP ID + 密钥 | 需在百度翻译开放平台注册 |

引擎切换后迷你窗口中已有文字会自动重新翻译。

### 5. 剪贴板监听

- 迷你窗口可见时，自动检测剪贴板新文本并填入原文区域
- 400ms 轮询间隔

### 6. 历史记录

- 翻译完成后自动记录原文、译文、引擎、语种、时间
- 超过 3 天的记录自动清理
- 与最新记录内容完全相同的翻译自动去重
- 存储目录可在设置中自定义（默认 `%APPDATA%/TranslationTool`）
- 最大 500 条记录
- 入口：托盘菜单「翻译历史」、迷你窗口底部「历史」按钮、完整窗口工具栏「历史记录」按钮

### 7. 历史记录窗口

- 卡片式列表展示翻译历史
- 支持关键词搜索原文或译文
- 每条记录可复制原文/译文（点击后显示「已复制」反馈）
- 可删除单条或清空全部

### 8. 设置窗口

- **翻译引擎**：配置百度翻译 APP ID/密钥，支持一键测试连接
- **OCR 设置**：Windows 系统 OCR 说明（免费，无需配置）
- **快捷键**：可视化捕获自定义快捷键，支持冲突检测
- **通用**：自定义历史记录存储目录

### 9. 快捷键管理

| 功能 | 默认快捷键 | 说明 |
|---|---|---|
| 迷你窗口 | Ctrl+Shift+Q | 显示/隐藏 |
| 完整窗口 | Ctrl+Shift+W | 显示/隐藏（切换） |
| 截图翻译 | Ctrl+Shift+E | 启动截图选区 |

- 修改快捷键时自动检测内部冲突和系统冲突，冲突时拒绝并提示
- 保存设置后立即生效（无需重启）

### 10. 窗口交互

- 设置窗口和历史窗口打开时，点击翻译窗口不会阻塞
- 有待确认弹窗（如测试连接失败）时，点击其他页面弹窗抖动提醒，无法操作其他页面
- 无待确认弹窗时，切换到其他窗口自动最小化
- 设置窗口和历史窗口最小化后，再次点击按钮自动从任务栏恢复

## 系统要求

- Windows 10 / 11
- Python 3.8+
- 主要依赖：PySide6、deep-translator、requests、pillow、winrt（Windows OCR）

## 数据存储

- 配置文件：`%APPDATA%/TranslationTool/config.json`
- 历史记录：`%APPDATA%/TranslationTool/history.json`（默认，可在设置中修改）
